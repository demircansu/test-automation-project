# Test Automation Project (SauceDemo Playwright Testleri)\n\nBu proje, SauceDemo e-ticaret sitesinin Playwright ile test otomasyonudur.\n\n## Özellikler\n- Page Object Model (POM)\n- API testleri\n- UI testleri (login, cart, checkout)\n\n## Kurulum\n```bash\nnpm install\n```\n\n## Test Çalıştırma\n```bash\nnpx playwright test\n```\n\n### Headed Mode\n```bash\nnpx playwright test --headed\n```\n\n### Tek Test\n```bash\nnpx playwright test tests/login.spec.ts\n```\n\n## Raporlar\n`npx playwright show-report`\n\n## Yapı\n- `pages/`: Page objects\n- `tests/`: Test dosyaları\n- `fixtures/`: Base test\n\nGitHub: https://github.com/demircansu/test-automation-project
